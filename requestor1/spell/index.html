Gmail like spell checker for your own application.

The most simple configuration

  • Decorate the text area

Code

var googie1 = new GoogieSpell("googiespell/", "https://www.google.com/tbproxy/spell?lang=");
googie1.decorateTextarea("ta1");

A little more advance configuration

  • Decorate the text area
  • Use another container
  • Support only Danish and English
  • Don't show close buttons

Code

var googie2 = new GoogieSpell("googiespell/", "https://www.google.com/tbproxy/spell?lang=");
googie2.setLanguages({'da': 'Danish', 'en': 'English'});
googie2.dontUseCloseButtons();
googie2.setSpellContainer("spell_container");
googie2.decorateTextarea("ta2");

Interface in Greek

  • GoogieSpell interface in Greek

Code

var googie4 = new GoogieSpell("googiespell/", "https://www.google.com/tbproxy/spell?lang=");
googie4.lang_chck_spell = "Έλεγχος ορθογραφίας";
googie4.lang_rsm_edt = "Επιστροφή στην επεξεργασία";
googie4.lang_close = "Κλείσιμο";
googie4.lang_no_error_found = "Δε βρέθηκαν ορθογραφικά λάθη";
googie4.lang_revert = "Επιστροφή σε";
googie4.setSpellContainer("spell_container4");
googie4.decorateTextarea("ta4");
